# BASH-Shell-Scripts

A practical repository to store, share, and reuse useful BASH shell scripts for automation, system administration, and workflow optimization on Unix/Linux environments.

---

## Overview

This collection helps sysadmins, DevOps engineers, and users streamline daily operations with scripts for file management, system info, networking, process automation, and batch tasks. Each script is well-commented for clarity and customization.

---

## Contents

- System information and monitoring
- File and directory manipulation
- User and permission management
- Text and log processing
- READ CSV File
- Miscellaneous helper scripts

---

## Getting Started

1. **Clone the repository**

git clone https://github.com/Rohan-Repo/BASH-Shell-Scripts.git

2. **Open and edit `.sh` scripts in your preferred editor (vim, nano, VS Code, etc.)**
3. **Make executable (if needed):**

chmod +x scriptname.sh

4. **Run scripts from the terminal:**

./scriptname.sh


---

## Prerequisites

- Bash shell (Linux, MacOS, WSL, or Unix)
- Root/user privileges for some operations
